# 🚀 CodeBuddy Notes

CodeBuddy Notes is a modern web application that helps students generate, organize, and manage programming notes efficiently.

## 📌 Features

* Generate Programming Notes
* Search Notes Instantly
* Add Notes to Favorites
* Delete Notes
* View Recent Notes
* Dashboard Statistics
* Dark Mode Support
* LocalStorage Data Persistence
* Responsive Design
* Modern SaaS Dashboard UI

---

## 🛠 Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript (ES6)

### Storage

* Browser LocalStorage

* Font Awesome

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/your-username/codebuddy-notes.git
```

### 2. Open Project

Open the project folder and launch:

```text
index.html
```

in your web browser.

No installation is required.

---

## 📖 How to Use

1. Select a programming language.
2. Enter a topic.
3. Click **Generate Notes**.
4. Notes will appear instantly.
5. Add important notes to Favorites.
6. Search notes using the search bar.
7. Toggle Dark Mode using the theme button.

---

## 💾 Data Storage

The application stores data locally using browser LocalStorage.

Stored items:

* Notes
* Favorites
* Theme Preferences

Data remains available after refreshing the page.
